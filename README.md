# projeto-programacao-a

Neste projeto será elaborado um programa para construir desenhos no estilo das apps Google Drawings e Libre Office Draws.

O projeto deverá utilizar o Tkinter, a biblioteca padrão de Python para construção de GUIs.

O projeto foi desenvolvido em equipes de três alunos Italo, Kassio e Hendric alunos de ciencia da computação da universidade federal de sergipe. 

Para testar, basta compilar o main.py
